# ✨ Features

🕸 **Interactive Web-Swing Navigation**  
Navigate the portfolio by swinging to glowing checkpoints as Miles Morales.

🌆 **Procedural 3D City**  
A stylized city environment generated with instanced meshes for performance and aesthetics.

🌙 **Dynamic Night Atmosphere**  
Custom lighting, glowing windows, and atmospheric effects inspired by the Spider-Verse aesthetic.

🎮 **Game-Like Portfolio Exploration**  
Explore an interactive 3D world instead of scrolling a traditional webpage.

🎵 **Immersive Audio System**  
Smooth transitions between cinematic intro music and an in-game soundtrack.

⚡ **Optimized Rendering**  
Efficient rendering using instancing, lazy loading, and optimized textures for a smooth experience.

# 🎮 Experience Flow

1️⃣ Land on the **Spider-Verse title screen** with an immersive glowing UI.

2️⃣ Click **Enter the Spider-Verse** to dive into the city.

3️⃣ The **3D neon city loads**, placing you in the shoes of Miles Morales.

4️⃣ Swing toward interactive checkpoints positioned across the cityscape.

5️⃣ Each checkpoint opens a themed panel:

• 🧑‍💻 **Origin Story** — About Me  
• 🚀 **Mission Log** — Projects  
• 🌐 **Socials** — Contact

# 📂 Project Structure

```text
src/
├── components/          # React Components
│   ├── Atmosphere.jsx      # Lighting, Sky, Moon, and Fog
│   ├── CityEnvironment.jsx # Procedural City Generation
│   ├── PlayerCamera.jsx    # Follow Camera & Cutscenes
│   ├── Checkpoints.jsx    # Interactive 3D Orbs
│   ├── ContentPanels.jsx   # UI Overlay Panels
│   ├── UIOverlay.jsx       # Global UI Elements
│   └── SpiderVerseText.jsx # Title Screen Text
├── App.jsx              # Main Entry Point
└── index.css           # Global Styles & Animations
```

# ⚙️ Installation

Clone the repository:
```bash
git clone https://github.com/RevanMidha/SpiderVerse-Portfolio.git
cd SpiderVerse-Portfolio
```

Install dependencies:
```bash
npm install
```

Run locally:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

---

# 🚀 Deployment

This project is deployed using **Vercel**.

To deploy manually run:
```bash
npm run build
```
Then upload the generated **dist/** folder to your hosting provider.
